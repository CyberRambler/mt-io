# 文本检测和识别

## 简介

为了能将一张图像中的多行文本识别出来，可以将该任务分为两步：

1. 文本检测：从图像中检测出所有文本行的矩形框，使用CTPN模型。
2. 文本识别：从文本行图像框中识别出文本，使用DenseNet模型。

## 训练

### 文本检测模型训练

CTPN训练使用的数据集格式与VOC数据集格式相同，目录格式如下：

    ```json
    - VOCdevkit
        - VOC2007
            - Annotations
            - ImageSets
            - JPEGImages
    ```
